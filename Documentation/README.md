## Criterio

- Las carpetas de modulo contienen estructura, tecnologias, flujo MVC, uso de POO, AJAX, Bootstrap y jQuery.
- `Documentation/API` contiene solo documentacion de endpoints para Category y DocumentType.
- `Documentation/IndicacionesProfesor` conserva el archivo original de indicaciones.
- `Documentation/General` conserva la documentacion general anterior del proyecto.
